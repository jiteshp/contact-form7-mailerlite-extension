Integrate Contact Form 7 With MailerLite email marketing.

== Description ==

Integrate Contact Form 7 With MailerLite email marketing. Automatically add form submission to a MailerLite subscriber, using its latest API.

= Features =
* Just add the API key, select a subscriber group and you're good to go.
* Use a different subscriber group for each contact form.
* Constantly updated plugin with the latest MailerLite API.

= Requirements =
* Self hosted WordPress.org
* Contact Form 7
* [MailerLite Account](http://www.mailerlite.com/a/hqq8vgczra "Sign up")

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload the plugin files to the `/wp-content/plugins/plugin-name` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress.
1. Use the 'Contact &rdquo; MailerLite Settings' screen to set the MailerLite API key.


All you need to do now is select a subscriber group from the 'MailerLite' tab in your Contact Form 7 Editor Screen. The subscriber email address is picked up from the *your-email* tag. Please make sure you have it in your form.
